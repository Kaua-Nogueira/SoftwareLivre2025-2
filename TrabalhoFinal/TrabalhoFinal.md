### Brave Voice Brasil

O **Brave Voice Brasil** é um plugin experimental desenvolvido para o navegador **Brave**, com o objetivo de realizar a **leitura de textos em voz alta (Text-to-Speech)**, priorizando vozes em **português brasileiro (pt-BR)** e oferecendo uma experiência de leitura mais fluida e acessível.

Seu diferencial é que, em vez de depender de APIs externas pagas (Google Cloud, Azure, etc.), o plugin usa a Web Speech API integrada no navegador. Isso reduz dependências externas, melhora privacidade e minimiza custos.

Para instalar e utilizar, basta assistir o vídeo abaixo:
